# Timing - Reverse Calendar

A simple app to log what you did instead of planning ahead.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Features

- **Log Page**: Record activities you completed
- **Timeline Page**: View activities in a vertical timeline
- **Analytics Page**: See time distribution in a pie chart

All data is stored in your browser's localStorage.
